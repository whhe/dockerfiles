# oblogproxy

The tools to build a Docker environment for [oblogproxy](https://github.com/oceanbase/oblogproxy). Referencing [obce-mini](https://github.com/oceanbase/oceanbase/tree/master/tools/docker/mini).

https://hub.docker.com/repository/docker/whhe/oblogproxy

Here are some versions:

- `ob-mini-local`: Deploy with an OceanBase cluster using mini-local config.
- `standalone`: Deploy a log proxy standalone server.

See more details at the subdirectories.
